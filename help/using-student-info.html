The Student table contains the information about the student that is required to write a recommendation. Additionally, there is a field for the professor to add comments about the student that can be referred to when the recommendation is writen.

Student Information Field Definitions

The Student first name, last name and email address are required fields. The profrssor comments qre optional; and, if present are displayed on the initial recommendlate.

  • Name Prefix (Optional) (i.e., Mr., Mrs., Ms.).
  • Student First Name Required
  • Student Last Name Required
  • Initial (Optional)
  • Suffix (Optional) (i.e., Jr., Sr., II, III).
  • Student Email (Required( The email address of the student.
  • Student School Student School
  • Date Student Requested Not modifable
  • Statement of intent, Point of view
  • Professor Comments Comments, concerning the student..
  • Timestamp Not modifiable. Timestamp of the last update to the record.
