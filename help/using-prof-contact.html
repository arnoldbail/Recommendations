The Professor Contacts table is used by the professor to save and access school contact resources. Typically, a student won't have the contact information for a school that was requested. The school willl need to be contacted to get the contact information. For individual schools, this is stored on the Schools table for the student. But, occasionally, a central site is used by many schools. This requires a username and password, which the professor stores on the contact table.

Professor Contacts Field Definitions

Only the Site field is a required field.

  • Site The site name.
  • Site URL The url of the site.
  • Username The professor username to login the site
  • Password The password for the username
  • Notess Any notes about the site
  • Timestamp Not modifiable. Timestamp of the last update to the record.
